HTML5 標準骨架 (Structure)

這是 HTML 的基本結構範例。你可以透過「檢視網頁原始碼」(Ctrl+U) 來查看完整的骨架。

關鍵組成部分：

  • <!DOCTYPE html>：告訴瀏覽器這是一個 HTML5 文件。
  • <html lang="zh-Hant">：根標籤，lang 指定網頁語言。
  • <head>：包含網頁的元數據 (Metadata)，如標題、編碼、樣式表連結，內容不會顯示在網頁主體。
  • <body>：包含所有可見的網頁內容（文字、圖片、連結等）。

小提醒： 在 VS Code 中，你可以輸入 ! 然後按 Enter，就會自動生成這樣的骨架！
