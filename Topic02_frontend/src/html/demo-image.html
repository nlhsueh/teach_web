圖片標籤 (Images)

1. 基本圖片引用

標籤：<img src="../img/virtuosi_hall.png" width="300px" alt="逢甲共善樓">

2. 網路隨機圖片 (Picsum)

標籤：<img src="https://picsum.photos/300/200" alt="隨機圖片">

3. 圓角樣式圖片 (搭配 CSS)

標籤：<img src="..." style="border-radius: 50%;">

重要屬性：

  • src：來源路徑 (URL 或 本地路徑)
  • alt：替代文字 (當圖片加載失敗或螢幕閱讀器讀取時使用)
  • width / height：寬高設定
