CSS 單位展示

1. Pixels (px) - 絕對單位

我是 24px 的文字，寬度 300px

固定大小，最精確但不具備彈性。

2. em - 相對單位 (相對於父元素)

我是父元素 (font-size: 20px)
我是子元素 (font-size: 1.5em = 30px)

基準點是父元素的字體大小。容易產生層層相乘的「複利效應」。

3. rem - 相對單位 (相對於根元素 html)

我是 1.5rem 的文字 (16px * 1.5 = 24px)

基準點永遠是根元素 (通常是 16px)。目前網頁設計的主流建議單位。

4. vw / vh - 相對單位 (相對於視窗 Viewport)

我的寬度永遠是視窗寬度的 50% (50vw)

隨著瀏覽器視窗大小即時縮放。常用於大標題或滿版佈局。
