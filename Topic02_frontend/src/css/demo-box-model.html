CSS 盒模型 (Box Model)

1. 盒模型元件

每個 HTML 元素都可以看作是一個盒子，包含：Content, Padding, Border, Margin。

我是內容
  • Content (內容): 文字或圖片所在處。
  • Padding (內邊距): 內容與邊框之間的距離。
  • Border (邊框): 包圍內容與內邊距。
  • Margin (外邊距): 盒子與其他元素間的距離。

2. box-sizing 比較

設定寬度均為 200px，內邊距 20px，邊框 10px。

Content Box
box-sizing: content-box;
總寬度 = 200 + 40 + 20 = 260px
Border Box
box-sizing: border-box;
總寬度 = 固定 200px
(內邊距與邊框會向內推)
