2. 多欄佈局 (Columns)

CSS 的 columns 屬性可以輕鬆實現像報紙一樣的多欄排版。你只需要指定欄數（如 3 欄），內容就會自動流向下一欄。 這在展示長篇文章或列表時非常有用，能有效利用水平空間。我們還可以使用 column-gap 來調整欄間距， 以及 column-rule 來加上分隔線。
