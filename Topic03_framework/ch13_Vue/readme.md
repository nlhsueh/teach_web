

# Learning Vue

* Exercise
  * `bmi`: requirements for a bmi exercise, and some students demo
  * `lab01a, lab01b`: based on w3school, learn vue basic concept and application.
  * `olympic`: a simple olympic web site to show the banner (as carousel) and the medals for each country. Version `paris00` - `paris06` for learning differnt Vue concepts.
  * olympic01-vue: using Vue SFC architecture. The futher version is placed in different branches. Please checkout `olympic02` - `olympic07` for different functions. [Demo](https://nlhsueh.github.io/vue-olympic) is the demo for version 07.
* `vue-basic`
  * based on W3school tutorial, including bind, if, for, event, on, method, event modifier, form model, computed, watch...
* `vue-sfc`
  * based on W3school tutorial about Single File Component (SFC), the main App.vue will call compoenents. 
  * demo01 (part01-04): basic component, prop, for, emit
  * demo05: fallthrough, scoped, local, slot, dynamic, http fetch
  * demo12: inject/provide
