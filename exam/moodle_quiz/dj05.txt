

::modelform:: [plain] 關於 ModelForm, 以下何者錯誤 {
~ [plain] 繼承自 form.ModelForm
= [plain] 每個 Model 都必須設計一個 ModelForm 才能輸入資料
~ [plain] 用來設定一個資料 Model 在輸入表單時的設計
~ [plain] 可以 class Meta 來設定個欄位要呈現的標題
}

::POST:: [plain] 關於 request method 為 POST, 以下何者為錯 {
= [plain] POST 表示資料要公佈在網站上
~ [plain] POST 表示資料要寫入儲存
~ [plain] PersonModelForm(request.POST) 表示依據表單填寫的內容建立一個 PersonModelForm
~ [plain] 在 html <form> 的標籤中，用 method\='POST' 來設定
}


::formAs:: [plain] ModelForm 中，有一些方法可以輸出 html 的介面，以下何者不是 {
~ [plain] f.as_ul()
~ [plain] f.as_p()
~ [plain] f.as_table()
= [plain] f.as_view()
}
